# download_video作用  
有部分的视频网站无法使用youtube-download进行数据下载，因此编写脚本对单独网页进行破解，这块的操作可能不适用，但是能提供一点思路

#### 流程图  
![image](https://github.com/yuguolong/download_video/blob/main/%E7%A8%8B%E5%BA%8F%E6%B5%81%E7%A8%8B%E5%9B%BE.jpg)

#### 快速开始  
配置 网页下载脚本.py 参数
需要输入的参数:  
url:        网页地址  
save_path:  保存地址  

注意:cookie参数需要自行设置
